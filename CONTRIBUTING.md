# 贡献指南

感谢你对物流路径规划系统的关注！

## 🤝 如何贡献

### 报告问题

1. 查看 Issues 确认问题未被报告
2. 创建新 Issue，包含：
   - 问题描述
   - 复现步骤
   - 期望行为
   - 截图

### 提交代码

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "feat: 添加某功能"`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

## 📝 代码规范

- Python: 遵循 PEP 8
- Vue: 使用组合式 API
- 提交信息: Conventional Commits

## ✅ 检查清单

- [ ] 代码能正常运行
- [ ] 遵循代码风格
- [ ] 添加必要注释
- [ ] 更新相关文档
