## 💡 关键决策记录

### 为什么选择 Vue 3？
- 小宇熟悉 Vue
- 组合式 API 更灵活
- Element Plus 组件丰富

### 为什么选择 Flask？
- 轻量级，快速开发
- Python 生态丰富
- 适合 ML 集成

### 为什么用 SQLite？
- 无需额外部署
- 适合中小规模
- 演示方便

### 为什么开发微信小程序？
- 司机需要移动端
- 微信生态完善
- 开发效率高

## 📝 经验教训

1. **需求变更要控制**：早期架构要灵活
2. **测试数据很重要**：生成器节省大量时间
3. **文档同步更新**：避免后期补文档
4. **部署要提前测试**：本地跑通不等于服务器能跑
