## 🎯 演示流程（建议 10-15 分钟）

### 第一部分：项目介绍（2分钟）

#### 项目背景
- 物流企业运输效率优化需求
- 路径规划 + 成本控制 + 实时监控

#### 核心价值
- 🚚 降低运输成本 15-30%
- ⚡ 提高配送效率 20%
- 📊 数据驱动决策

#### 技术亮点
- 4种智能调度算法
- ML需求预测（LSTM+Prophet）
- 实时WebSocket推送
- 3D地球可视化

---

### 第二部分：核心功能演示（8分钟）

#### 1. 数据仪表盘（1分钟）

**访问路径**: 首页 `/dashboard`

**演示要点**:
- 关键指标卡片（订单、车辆、成本）
- 趋势图表
- 预警提示

**解说词**:
> "这是系统的数据仪表盘，可以看到订单总数、车辆状态、今日成本等关键指标，帮助管理者快速掌握运营状况。"

---

#### 2. 智能调度（2分钟）

**访问路径**: 智能调度 `/dispatch`

**演示步骤**:
1. 选择优化目标（距离/时间/成本）
2. 调节权重滑块
3. 点击「开始调度」
4. 查看调度结果

**解说词**:
> "这是我们的智能调度系统，采用遗传算法自动分配订单到车辆。你可以调节权重，比如更看重成本还是时间，系统会给出最优方案。"

**算法亮点**:
- 遗传算法 - 自动分配
- 蚁群算法 - 路径优化
- 粒子群 - 多目标平衡
- 深度强化学习 - 动态调整

---

#### 3. 地图可视化（1.5分钟）

**解说词**:
> "地图页面集成了高德地图，可以实时查看车辆位置、运输轨迹，还支持轨迹回放功能。"

### 第四部分：技术架构说明（2分钟）

#### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + Element Plus | 响应式UI |
| 后端 | Flask + SQLAlchemy | REST API |
| 实时 | WebSocket | 毫秒推送 |
| 算法 | Python | 4种优化算法 |
| ML | TensorFlow | LSTM预测 |
| 移动端 | 微信小程序 | 司机端 |

#### 部署方式

**Docker 一键部署**:
```bash
docker-compose up -d
```

## 🚀 快速部署指南

### Docker 部署

```bash
# 1. 克隆项目
git clone https://github.com/Tyranitar-design/logistics-route-planning.git
cd logistics-route-planning

# 2. 一键启动
docker-compose up -d

# 3. 访问系统
# 前端: http://localhost:8080
# 后端: http://localhost:5000

# 4. 登录
# 账号: admin
# 密码: admin123
```
